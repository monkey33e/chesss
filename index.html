AI 역사

인공지능의 발전 과정을 시대순으로 살펴보세요

1950년대 - AI의 탄생

앨런 튜링의 튜링 테스트와 AI 개념의 등장

  • • 1950: 튜링 테스트 제안
  • • 1956: 다트머스 회의
  • • "인공지능" 용어 탄생

1960-70년대 - 초기 발전

전문가 시스템과 초기 신경망 연구

  • • 1957: 퍼셉트론 개발
  • • 1960년대: 전문가 시스템
  • • LISP 프로그래밍 언어

1980년대 - AI 겨울

기대에 못 미치는 성과로 인한 침체기

  • • 연구 자금 감소
  • • 한계 인식
  • • 역전파 알고리즘 개발

1990년대 - 머신러닝 부상

통계적 접근법과 실용적 AI 시스템

  • • 1997: Deep Blue vs 카스파로프
  • • SVM, 랜덤 포레스트
  • • 인터넷과 데이터 증가

2000년대 - 빅데이터 시대

대용량 데이터와 컴퓨팅 파워 증가

  • • 구글, 페이스북 등장
  • • 웹 데이터 활용
  • • GPU 컴퓨팅 시작

2010년대 - 딥러닝 혁명

신경망의 부활과 획기적 성과

  • • 2012: AlexNet ImageNet 우승
  • • CNN, RNN 발전
  • • 2016: AlphaGo vs 이세돌

2017-2020 - Transformer 시대

어텐션 메커니즘과 대규모 언어 모델

  • • 2017: Transformer 논문
  • • 2018: BERT 발표
  • • 2019: GPT-2 공개

2020년대 - 생성 AI 시대

ChatGPT와 생성형 AI의 대중화

  • • 2020: GPT-3 발표
  • • 2022: ChatGPT 출시
  • • 2023: GPT-4, 멀티모달 AI
